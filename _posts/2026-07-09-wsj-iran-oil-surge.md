## 1｜头版聚焦（Front Page）

### 1. Trump 宣布伊朗停火结束——油价暴涨 5.2% 至 $73.52

- **核心事实**：Trump 在安卡拉北约峰会期间宣布与伊朗的停火协议"结束"，随后美军对伊朗发动新一轮打击。打击范围比周二更广——不仅锁定此前的 Bandar Abbas 和 Sirik，还涉及波斯湾周围导弹和无人机存储区以及军用桥梁。油价单日暴涨 **5.2%**，为 5 月以来最大单日涨幅。Dow -1.1%，S&P -0.3%，但 Nasdaq +0.2%。
- **WSJ 独家角度**：文章详细报道了美军打击战略的升级（从"防御性"转向"惩罚性"），以及 Trump 对先前商船攻击的回应逻辑。
- **市场影响**：油价 $73.52 重新点燃通胀担忧。10Y 收益率升至 **4.567%**。黄金暴跌 $74（$4,071）——risk-off 但美元的避险属性同时压制黄金。

### 2. Obamacare 保险公司寻求 2027 年大幅涨费——两年连涨 35%+28%

- **核心事实**：Centene 在华盛顿州寻求 **28%** 的保费上涨（2026 年已涨 35%）。伊利诺伊州的 Blue Cross 要求 **15%**（2026 年已涨 28%）。KFF 的 Cox："将两年加在一起看是惊人的跳跃。"
- **为什么重要**：医疗保费暴涨将成为中期选举的核心议题。联邦补贴减少 + 医药和医院成本持续上升 + 更健康的人退出市场（逆向选择） = ACA 市场的恶性循环。

### 3. Blue Origin 寻求 $100 亿融资——估值 $1,300 亿

- **核心事实**：Jeff Bezos 的太空公司正在策划一轮 **$100 亿**的融资，估值达 **$1,300 亿**。这是私人航天行业有史以来最大规模的融资之一。
- **为什么重要**：Blue Origin 的估值 ($130B) 相对于 SpaceX（$200B+ 峰值）仍有一倍差距。New Glenn 火箭的持续进展和 NASA 合同是估值支撑。**这是"太空竞赛 2.0"中 Bezos 对 Musk 的反击。**

### 4. Trump 家族加密项目出售核心业务——已向家族输送超 $5 亿

- **核心事实**：Trump 家族的加密风险投资——此前已向家族输送超过 **$5 亿**——正在洽谈将其核心业务出售给区块链技术公司 Perpetuals.com。延续昨日 WSJ 披露的 Trump 加密帝国获利 $8 亿的故事线。
- **为什么重要**：这是 Trump 家族商业帝国的"退出策略"：在加密热潮高点套现。**对投资者而言，这强化了"名人加密项目 = 创始人先跑"的叙事。**

### 5. Apple 将在未来 5 年向 Broadcom 支出 $300 亿+——用于美国造芯片

- **核心事实**：Apple 计划在未来 5 年内向 Broadcom 支出 **$300 亿+**，用于美国制造的芯片。这是 Apple 在"美国制造"和"芯片多元化"战略下的重大承诺。
- **投资含义**：Broadcom (AVGO) 继昨天 Apple 续约至 2031 后再获确认——两个独立来源（定制芯片 + $300 亿支出）同时确认 Avago 的 Apple 收入可见性极高。

### 6. 中国指控 Anthropic 的 Claude Code 存在"安全后门"

- **核心事实**：中国官方宣布在 Anthropic 的 Claude Code 中发现了"安全后门漏洞"。Claude Code 是 Anthropic 面向开发者的 AI 编程工具。
- **为什么重要**：这是中美 AI 地缘政治紧张的最新升级——类似华为/字节跳动被指控的逻辑反转了（中国指控美国 AI 公司存在安全风险）。可能影响 Anthropic 的中国市场准入，并引发其他政府对 Claude Code 的审查。

### 7. Trump 支持乌克兰打击俄罗斯纵深——重大政策转向

- **核心事实**：Trump 在安卡拉北约峰会上与 Zelensky 会面时说："这是一种升级，但也是可能帮助结束战争的升级。"他还表示将考虑授予乌克兰生产 **Patriot 拦截器** 的许可。此前乌克兰 Patriot 弹已基本告罄。乌克兰还在推动获得在俄罗斯上空使用 Starlink 的权限。
- **为什么重要**：这是 Trump 在乌克兰问题上最强有力的亲乌表态。**从"24 小时结束战争"到"支持打击俄罗斯纵深"——Trump 乌克兰政策的 180 度转变有可能改变战争格局。** 对国防股（Patriot 制造商 RTX）是明确利好。

### 8. Costco 的"百万美元收银员"

- **特色报道**：60 岁的 Costco 收银员 Tony Barzar 时薪 $32.90，401(k) 超过 $100 万，医疗自付 $15/次，两年前和家人买了带泳池的三卧室别墅，去过欧洲两次。Costco 执行层："长期员工是我们秘密武器。"
- **为什么重要**：这是 Costco 商业模式（高工资 → 低 turnover → 高生产力 → 高利润）的生动案例。在 Walmart 降价的同时，Costco 证明高薪零售同样可行。
